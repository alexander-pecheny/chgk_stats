# count_q_rating  

## Что это?
Скрипт группирует вопросы в зависимости от доли их взятий на турнире.  

| Группа | Доля взятий |
|--------|-------------|
| a | менее 0.1 |
| b | 0.1–0.33 |
| c | 0.33–0.66 |
| d | 0.66–0.9 |
| e | более 0.9 |
| гроб | нет взятий |
| снят | вопрос снят |
> Большее значение в диапазоне указывается включительно  

Результатом работы скрипта становится таблица со статистикой взятий вопросов по группам для команд и распределение вопросов по группам на турнире в целом.  

## Использование

### Общий случай
Через командную строку запустить файл, указав ID турнира, для которого нужно провести расчёт.  
Например, `count_q_rating.py 11740`

### Отображение флагов в таблице
Чтобы отобразить в первом столбце флаги команд на турнрире, нужно использовать `--add_flags`.  
По умолчанию будут выведены все. Чтобы вывести только конкретные, укажите через запятую, какие именно: `count_q_rating.py 11740 --add_flags 7,59`  
Флаги добавляются по их id (см. [таблицу ниже](#Таблица-флагов)).  

### Фильтрация по флагам
Если нужно, например, посчитать статистику только по командам студенческого зачёта, можно использовать `--flags 4`. Можно фильтровать по нескольким флагам (используется логическое «ИЛИ» — команда учитывается, если у неё есть хотя бы один флаг из указанных).  
Флаги добавляются по их id (см. [таблицу ниже](#Таблица-флагов))
